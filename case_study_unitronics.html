Автоматизация санпропускника НС NEROS

Самостоятельная разработка логики на ПЛК UNITRONICS JZ20-31 для управления гигиеническим шлюзом.

UNITRONICS JZ20-31 Ladder Logic (LD) Автоматизация КИПиА

Обзор проекта

Санпропускник NEROS на входе в "чистую зону" производства работал некорректно / в ручном режиме. Это создавало риски нарушения санитарных норм и приводило к задержкам при проходе персонала. Задача состояла в том, чтобы полностью автоматизировать его работу, используя имеющийся в наличии ПЛК UNITRONICS JZ20-31.

Компоненты и Задачи

Проект включал в себя реверс-инжиниринг работы механики санпропускника и программирование ПЛК с нуля.

Анализ и Подключение

Задача: Изучить логику работы санпропускника и подключить все датчики и исполнительные механизмы.

Действия:

  • Идентификация датчиков (оптические датчики рук, датчик турникета).
  • Подключение исполнительных механизмов (подача дез. средства, управление турникетом).
  • Физический монтаж и подключение ПЛК UNITRONICS в шкаф управления.

Программирование Логики (LD)

Задача: Написать программу на Ladder Logic (LD) для реализации требуемого цикла работы.

Действия:

  • Создание программы в ПО U90 Ladder.
  • Реализация таймеров для дозирования дез. средства.
  • Программирование пошагового алгоритма (вход -> дезинфекция -> разблокировка турникета).
  • Настройка HMI (встроенный дисплей ПЛК) для индикации статуса.

Техническая реализация

Алгоритм работы (Язык LD)

Логика работы была реализована на языке релейных диаграмм (LD), который идеально подходит для таких пошаговых задач: 

| |
|---[ ]---|---[ ]---|----(SET)----|
|  DI_Hands_In  |  NOT_Block  |  DO_Dez_Pump  |  (Подать дез. средство)
| |
| |
|---[ ]---|----(TMR)----|
|  DO_Dez_Pump  |  T_Dez_Time  |  (Таймер дезинфекции 3 сек)
| |   ( 3s )  |
| |
|---[ ]---|---[ ]---|----(SET)----|
|  T_Dez_Time   |  NOT_Block  |  DO_Turniket  |  (Разблокировать турникет)
| |
| |
|---[ ]---|----(RST)----|
|  DI_Turniket  |  DO_Dez_Pump  |  (Сброс при проходе)
| |
| |---[ ]---|----(RST)----|
|  DI_Turniket  |  DO_Turniket  |
| |

(Это упрощенный пример логики)

Результаты и выводы

Ключевой вклад:

Самостоятельное решение задачи автоматизации "под ключ" на нетипичном для основной работы оборудовании (UNITRONICS). Проект демонстрирует гибкость, умение быстро осваивать новое ПО (U90 Ladder) и языки (LD), а также находить эффективные решения для локальных производственных задач.

  • Повышение уровня гигиены: Гарантированный цикл дезинфекции рук для всего персонала, входящего в "чистую зону".
  • Автоматизация процесса: Исключен человеческий фактор и ручное управление, ускорен проход персонала.
  • Расширение компетенций: Получен успешный опыт программирования ПЛК UNITRONICS, что доказывает способность работать с контроллерами различных производителей.
Этот проект является частью портфолио Людвига К.А.
